Documentación Oficial

La documentación oficial de Node.js es el recurso más completo para aprender a utilizar esta tecnología. Aquí encontrarás guías, referencias de API y tutoriales para todos los niveles de experiencia.

La última versión estable de Node.js es la v20.11.1 LTS. Se recomienda utilizar siempre la versión LTS (Long Term Support) para proyectos en producción.

Referencia de API

Node.js incluye una amplia variedad de módulos incorporados que puedes utilizar sin instalación adicional. Aquí están algunos de los más importantes:

Módulo Descripción Estado
fs Sistema de archivos: leer, escribir y manipular archivos y directorios Estable
http/https Creación de servidores web y clientes HTTP Estable
path Utilidades para trabajar con rutas de archivos y directorios Estable
events API para trabajar con eventos y emisores de eventos Estable
stream Interfaz para trabajar con datos de streaming Estable
esm Soporte para módulos ECMAScript (ES) En evolución
