¿Qué es Node.js?

Node.js es un entorno en tiempo de ejecución multiplataforma, de código abierto, para la capa del servidor basado en el lenguaje de programación JavaScript. Es asíncrono, con E/S de datos en una arquitectura orientada a eventos y basado en el motor V8 de Google.

Fue creado con el enfoque de ser útil en la creación de programas de red altamente escalables, como por ejemplo, servidores web. Fue desarrollado por Ryan Dahl en 2009 y su evolución está apadrinada por la empresa Joyent.

A diferencia de la mayoría del código JavaScript, Node.js no se ejecuta en un navegador, sino en el servidor, permitiendo un desarrollo homogéneo entre cliente y servidor.

Características Principales

Concurrencia

Funciona con un modelo de evaluación de un único hilo de ejecución, usando entradas y salidas asíncronas que pueden ejecutarse concurrentemente en un número de hasta cientos de miles.

Motor V8

Utiliza V8, el entorno de ejecución para JavaScript creado para Google Chrome. Compila el código fuente JavaScript en código de máquina en lugar de interpretarlo en tiempo real.

Sistema de Módulos

Incorpora módulos básicos compilados en el propio binario y permite utilizar módulos desarrollados por terceros, gestionados fácilmente con el Node Package Manager (npm).

Aspectos Técnicos

Bucle de Eventos

Node.js se registra con el sistema operativo y cada vez que un cliente establece una conexión se ejecuta un callback. Dentro del entorno de ejecución, cada conexión recibe una pequeña asignación de espacio de memoria dinámico, sin tener que crear un hilo de ejecución.

Desarrollo Homogéneo

Node.js puede ser combinado con bases de datos documentales (MongoDB, CouchDB) y relacionales (MySQL, PostgreSQL), lo que permite desarrollar en un entorno JavaScript unificado. Facilita la reutilización de código del mismo modelo de interfaz entre el lado del cliente y el servidor.

Historial de Versiones

Las nuevas versiones principales de Node.js se liberan desde la rama máster de GitHub cada seis meses. Las versiones pares se lanzan en abril, y las impares en octubre. Las versiones pares anteriores pasan a soporte a largo plazo (LTS).
